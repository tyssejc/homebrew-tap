# Tyssejc Tap

## How do I install these formulae?

`brew install tyssejc/tap/<formula>`

Or `brew tap tyssejc/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "tyssejc/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
